AERSMine

I worked with a senior developer at Cincinnati Children's Hospital to revamp and develop AERS, a data-mining web application. The app allows users to analyze drug, reactions, and indications information from millions of patient reports. Medical professionals and data scientists use the app to simulate drug trials using existing data, which can help to replace the need for extensive manual trials.

  • Employer: Cincinnati Children's Hospital Medical Center
  • Dates: 2013-2015
  • Service: Front-End Development

March Madness Bracket Generator

A friend and I started a side project during the 2016 NCAA Men's Basketball tournament. We wanted to see how well we could match a the 64-team bracket using a probabalistic algorithm. Some of our work can be seen at this Github repository. We used our algorithm (colloquially known as "log2") to correctly predict the winner of both 2016 and 2017 March Madness tournaments!

  • Dates: Spring 2016

A Massively Parallel Algorithm for Cell Classification Using CUDA

My Master's Thesis at the University of Cincinnati used various Machine Learning approaches to highly optimize a cell classification algorithm. I parallelized a Gaussian Bayesian classifier using CUDA, a GPU computing platform developed by NVIDIA which allows thousands of computing threads to work together to speed up complex mathematical calculations. This classifier has been proven to be highly accurate for this type of problem. My parallel algorithm improved on the serial version by over 700%.

  • Dates: 2013-2015
